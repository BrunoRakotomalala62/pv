## 📚 Comment utiliser le bot

Utilisez la commande `help` pour découvrir toutes les fonctionnalités disponibles !

```
help           → Liste des commandes
help <numéro>  → Affiche une page spécifique
next / >       → Page suivante
prev / <       → Page précédente
```
